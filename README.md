# 🚆 BookMyTrain - Java Console Project

A simple **IRCTC-style train ticket booking system** built in Java. This project simulates booking, user registration, and train management functionalities.

---

## 📌 Features

- ✅ Add & list trains
- ✅ Register users
- ✅ Book train tickets
- ✅ Check seat availability
- ✅ Generate unique PNR numbers
- ✅ View ticket details

---

## 🛠️ Tech Stack

- Language: Java
- Environment: Console-Based
- No external libraries used

---

## 📂 Project Structure

BookMyTrain/
├── Main.java // Entry point
├── IRCTCAPP.java // Application interface
├── User.java // User model
├── Train.java // Train model
├── Ticket.java // Ticket model
├── BookingService.java // Ticket booking logic
├── UserService.java // User management logic
